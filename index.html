Let's Start Typing

How To Take the Test:

The timer for the typing test will begin on your first keystroke. If you choose to take a test based on "words", you will choose a set number of words which you will type out as fast as you can. If you choose to take a test based on "time", there will be a set time limit, in which you can type as many words as possible. You can adjust the number of words/time limit to your preference. Good luck!
